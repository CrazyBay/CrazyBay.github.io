搭建个人博客的心得

发表于 2017-07-08 | 分类于 hexo | | 阅读次数

上学那会，自己不怎么爱记笔记，也没觉得记笔记有啥好的，大学期间更别提了，笔记少之又少，突然意识到自己该记点笔记是从刚工作实习那会开始，技术渣渣就不说了，但是还是要有一颗向上的心，于是实习期间就读书学习，但是学完之后总觉得自己没学多少东西，才感觉到记笔记的重要性了。不过使用云笔记之类的东西吧，自己又不太习惯，所以零零散散的记了一些东西。总得来说，记录不多。搭建这个博客是想记录一下自己的工作和生活中的成长，希望自己养成记录的习惯。
